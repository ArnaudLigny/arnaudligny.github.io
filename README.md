# narno.org

Source of my [home page](http://narno.org), generated with [PHPoole-library](https://github.com/Narno/PHPoole-library).

## install
```
$ git clone https://github.com/Narno/narno.github.com
$ git checkout source
$ composer install
```

## test locally
```
$ php build.php
```

## publish
```
$ php build.php -e=prod -p
```
